# cli 1.3.3

The `cli` tool for syr projects. Get Started Fast.

## Get started

```bash
npm install -g @syr/cli
```

## Usage

```bash
syr init MyProjectName
```

## Dependencies

* npm 5+


## Development Notes

* app is written in ESNext
* babel import aliases are used to alleviate relative importing
